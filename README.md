## カードの要素
カードに並べる要素をメインとサブと演出に分けて、気配多めで表現する。

### メインの情報

- 丸いアバター画像→これは必須
- 日本語名「わかどり」→必須
- 英字名「wakadori」→英語圏向けに必要？

### サブの情報

- リード文「イラストと技術、ふたつの顔で活動しています。」→もうちょい練りたい
- 見出し「Links」→このへんは考える余地あり
- リンク項目（Illustration / Tech の2本）→これは裏表に分離が良いかな？
- 裏面のキッカー「Code / UI」→いい感じ
- 裏面の見出し「つくる側の顔」→これも文言凝りたい
- 裏面リード文「実装メモやUIの試作をまとめています。」→もうちょい練りたい

### 演出

- カード形状、半透明、影
- 3Dの傾き・回転
- 斜めに走る光沢ハイライト
- カード下部の円形の窪み
- オーブ型トグル一式（外周／球体／光沢）
- 表面／裏面の切り替え構造
- スナップショット枠（プレースホルダー含む）

## 要素の方針

### リンク・遷移の考え方

* 外部リンクは「押すと遷移する」ことが一目で分かる表現にする
* 勝手にアプリへ飛ぶ不快感を避ける

### 深掘り体験（重要要素）

* 作品・コードなど、よく見てほしい要素は
  別カードがせり出す形で表示する
* PCでは自然に横並びになる

### 言語対応（JP / EN）

* 日本語：空気感・やわらかさ・余白担当
* 英語：意味・導線・行動内容の把握担当
* 完全な直訳はせず、役割に応じて表現を分ける

### ダークモード方針

* ライト基調をベースにダーク対応する
* 色の完全反転はしない
* ダークでは透明度をやや下げ、濁りを避ける
* テキストは純白を避け、目の負担を減らす
* 色は直接指定せず、意味ベースの変数で管理する
